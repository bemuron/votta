Votta - Your Voice, Your Vote

Votta is a simple online voting system that gets the basics of handling online elections done.
No fancy complicated, unnecessary additions. People simply vote and you can see the results.

Features
* Manage elections, candidates and elective positions.
* Create several elections and set it's status to active or draft.
* Set start and end dates for each election.
* The results of ongoing and completed elections can be seen.
* A voter can only vote once for a specific position.

Tech Stack
* Laravel 8 with Bootstrap 5
* MySql Database
